# Learning Deno

The Deno website [https://deno.land/](https://deno.land/)

## installation

Using shell(Linux or Mac):

`curl -fsSL https://deno.land/x/install/install.sh | sh`

using PowerShell(Windows):

`iwr https://deno.land/x/install/install.ps1 | iex`
